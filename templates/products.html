Products

The Zepto Analytics system is built up of three components, the Node, Hub and Engine..

Node

Nodes are small low power wireless units that collect data from a range of sensors

Hub

The Hub collects data from the nodes

It can be configured to either store the sensor data in its internal storage, or send it on to a remote server or cloud system

Engine

The Engine lets you monitor and analyse the data coming from the nodes, any time, any where.

Try out a demo here: Engine Demo
